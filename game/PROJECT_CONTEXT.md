# open//neo vs claw monkeys — Game Project Context

## Концепция

Браузерная аркадная 2D-игра в пиксельном стиле. Главный герой — **open//neo** (Нео из Матрицы в розовом плаще) — сражается против **claw monkeys** (злых обезьян с красными глазами), которые олицетворяют рутинные бизнес-задачи. Обезьяны стреляют в Нео пиксельными фаллосами. Нео кидает в ответ клешню-сюрикен 🦞.

Игра задумана как интерактивный промо-элемент для проекта **open//neo** (AI-ассистент/автоматизация). Победа = промокод на скидку 15%.

## Стек

- React (функциональные компоненты + hooks)
- Tailwind CSS (utility-классы, без компилятора — только базовые)
- Canvas API для рендера пиксельных спрайтов
- Чистый JS game loop через `requestAnimationFrame`
- Без внешних зависимостей (framer-motion, shadcn убраны)

## Персонажи и спрайты

### open//neo (Игрок)

- Спрайт: 26×46 пикселей
- Pixel scale: `MPS * 1.05` (на 5% больше обезьян)
- Палитра: чёрные волосы, кожа, синие отражения в очках, розово-пурпурный плащ Matrix-стиля с высоким воротником, тёмные штаны, чёрные ботинки
- Разворачивается по направлению движения (flipX)
- Розовый glow drop-shadow
- Мигает при неуязвимости (invuln)
- Подпись "open//neo" бирюзовым под спрайтом

### Claw Monkey (Враг)

- Спрайт: 28×28 пикселей
- Pixel scale: 3
- Палитра: коричневая шерсть (4 оттенка), розоватая кожа лица, красные злые глаза, белые белки, тёмный outline
- Большие круглые уши с розовой внутренностью
- Хвост растёт из нижней части спины, закручивается вверх вправо (как у настоящей обезьяны)
- Светлое пузико
- Разворачивается лицом к игроку
- Над каждой обезьяной — label с названием задачи
- Под обезьяной — статус "Стреляет" (оранжевый) / "Перезарядка" (зелёный)
- Зелёный glow-border в фазе перезарядки

### Снаряд обезьяны (PP_SPRITE)

- Спрайт: 18×10 пикселей, pixel scale: 2
- Пиксельный фаллос с яйцами у основания и головкой с ридж
- Палитра: розово-телесные тона (6 оттенков от `#f0c4c8` до `#7a3840`)
- Разворачивается по направлению полёта
- Также используется в экране проигрыша (куча вокруг Нео)

### Клешня Нео (Оружие)

- Эмодзи 🦞 (25px, на 15% больше стандартного)
- Вращается как сюрикен при полёте
- Запускается на уровне лба обезьяны (y-offset 80 от верха Нео)
- Убивает обезьян только в фазе "Перезарядка"

### Золотая клешня (Супероружие)

- Эмодзи 🦞 с CSS-фильтрами: `sepia + hue-rotate + brightness + saturate`
- Размер: 54×54px (на 50% больше стандартного), эмодзи 42px
- Пульсирующее золотое glow-кольцо (анимация `sPulse`)
- Вертикальный столб золотого света снизу
- Подпрыгивающая анимация (`sBob`)
- Подпись "SUPER" золотым сверху
- Спавнится каждые ~16 сек (±3 сек рандом), первая — через ~9 сек
- При подборе: убивает ВСЕХ врагов вне зависимости от фазы

## Эффекты

### Глитч-взрыв (при подборе золотой клешни)

- 24 мини-клешни разлетаются радиально от точки подбора
- Ещё 6 клешней из позиции каждого убитого врага
- Клешни: вращение, затухание, гравитация, золотой glow
- Экран: shake (14px амплитуда), RGB chromatic aberration (красный + зелёный split), scanlines overlay
- Золотая вспышка в первые 20% анимации
- Длительность: 600мс

### Мигание при попадании

- Нео мерцает opacity 0.35↔1 в течение 1200мс неуязвимости

## Геймплей

### Механики

- Движение: ← → / A D (8px/frame)
- Прыжок: ↑ / W / Space (начальная скорость -15, гравитация 0.82)
- Атака: Enter / F (кулдаун 520мс, анимация 170мс)
- Рестарт: R

### Враги

- На поле всегда минимум 2 обезьяны
- Фазы: "aiming" (стреляет по таймеру) → "reloading" (уязвима) → цикл
- Скорость и кулдаун масштабируются с количеством убийств
- Патрулируют горизонтально, разворачиваются у стен
- Имя задачи из списка 20 бизнес-задач

### Баланс

- 3 жизни у Нео
- 15 убийств для победы
- Скорость пуль врагов: 6.7 + kills * 0.08 (макс +1.4)
- Кулдаун врагов уменьшается с прогрессом (2500мс → 1400мс)
- Перезарядка врагов укорачивается (1800мс → 1000мс)

### Экраны

**Победа:** 🎁, "Поздравляем!", "СКИДКА 15%", промокод OPENNEO15

**Проигрыш:** Стилизован под дизайн сайта openclaw-setup (Share Tech Mono, magenta/cyan палитра, scanlines). Заголовок "Buried under ~~dicks~~ tasks" — "ick" глитчит с заменой на `@#$%&!?*<>` как hover на skill-карточках сайта, chromatic aberration cyan/magenta. Пирамида из ~70 пиксельных снарядов (PP_SPRITE) в форме как куча черепов — широкое основание, сужается кверху. На вершине — голова и воротник Нео (обрезанный спрайт, 19 строк, scale 4.2x). Подпись "The monkey work won this round." Кнопка "Try again" — залитая magenta с glow, как CTA на сайте. Хинт "[ press R to restart ]".

## Контекст продукта

Игра является промо-механикой для **open//neo** — проекта автоматизации бизнес-задач. Обезьяны символизируют рутинные задачи ("Разбор почты", "Контроль дедлайнов", "Обновление CRM" и т.д.), которые "атакуют" пользователя. open//neo борется с ними, автоматизируя процессы.

Победа в игре = промокод на скидку, что создаёт геймифицированную воронку привлечения.

## Технические заметки

- Все спрайты описаны как 2D-массивы с числовыми индексами палитры
- `drawSprite()` рендерит массив на canvas с поддержкой flipX и дробного pixel scale
- Game loop через `requestAnimationFrame` с delta-time capping (max 33ms)
- State хранится в React state + refs для доступа из game loop без stale closures
- Нет localStorage/sessionStorage (не поддерживается в Claude artifacts)
- Нет внешних анимационных библиотек — только CSS @keyframes для UI элементов
